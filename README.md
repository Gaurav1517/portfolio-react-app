# 🎨 Personal Portfolio – React App

Welcome to my **React-based portfolio project**, built to showcase my skills, projects, and experience as a developer. This app is lightweight, fast, and easily customizable.

---

## 📦 Features

- Built with **React** using `create-react-app`
- Fully responsive design
- Clean and modular code structure
- Easy to deploy and extend

---

## 🛠️ Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/Gaurav1517/portfolio-react-app.git

# Navigate into the project directory
cd portfolio-react-app

# Install project dependencies
npm install

# Start the development server
npm start
````

The app will automatically open in your browser at:
👉 **[http://localhost:3000](http://localhost:3000)**
